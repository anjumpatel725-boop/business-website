# Business Website

A fully responsive business website built using React.js with modern UI design.

## Live Website

https://business-website-theta-olive.vercel.app/

## Features

- Responsive design for all devices
- Mobile navigation menu
- Dark/light theme toggle
- Modern CSS Grid and Flexbox layouts
- Animated UI components
- Contact form validation
- Reusable React components
- Accessibility-friendly design

## Technologies Used

- React.js
- JavaScript
- CSS3
- Vite
- GitHub
- Vercel
